## 💡 Giới thiệu

Hệ thống đặt đồ ăn online.

Các bước cài đặt:

1. Clone dự án này về.

2. Cài đặt thư viện:

```sh
npm i
```

3. Chạy giao diện:

```sh
cd frontend
npm run dev
```

4. Chạy server:

```sh
cd backend
npm run dev
```
